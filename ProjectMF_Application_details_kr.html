이 프로젝트는 project MF의 UI/UX 업무와 3d 배경 업무에 지원하는 프로젝트 지원 포트폴리오입니다.

이 프로젝트는 블루아카이브의 팬게임인 ProjectMF에 지원하는 포트폴리오입니다. 지원한 분야는 UI/UX 부분과 3D 배경입니다.
이 프로젝트에는 다음과 같은 내용이 포함되어있습니다.
- 요청한 블루아카이브의 팬게임이라는 것을 한방에 알 수 있는 UI
- 기존 FPS의 UI를 따라 FPS 메뉴조작과 유사한 UX
- 블루아카이브의 UX와 유사하도록 마우스 클릭시 파티클 생성 및 드래그시 마우스 궤적을 위젯 도메인에 생성
- 기존 블루아카이브에 맞는 툰 쉐이딩에 적합한 3d 배경 모델
